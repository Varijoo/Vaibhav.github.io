# Vaibhav Portfolio (GitHub Pages)

This is a simple, recruiter-friendly portfolio site (single-page HTML/CSS).

## Quick start (publish on GitHub Pages)

1) Create a new repository named **Varijoo.github.io** (exactly).
2) Upload these files to the repo root:
   - index.html
   - styles.css
   - assets/ (folder)
3) In GitHub: **Settings → Pages**
   - Source: Deploy from branch
   - Branch: main / root
4) Your site will be live at: https://varijoo.github.io

## What to edit first

- `index.html`
  - Replace `YOUR_EMAIL_HERE` in the mailto link.
  - (Optional) Replace headline/subhead text.
- `assets/resume.pdf`
  - Replace with your actual resume PDF (keep the same filename).

## Optional: Custom domain
You can add a custom domain in Settings → Pages later.
