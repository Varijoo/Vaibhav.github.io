## Update Instructions (Quick)
### 1) Update Resume
Replace the file in:
`assets/resume.pdf`

Keep the filename exactly the same: `resume.pdf`

### 2) Update Email
In `index.html`, search for:
`YOUR_EMAIL_HERE`

Replace with your email inside:
`mailto:your@email.com`

### 3) Edit Content
Main text lives in:
`index.html`

Styling lives in:
`styles.css`
